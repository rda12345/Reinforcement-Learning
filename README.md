# Reinforcement Learning

The repository contains various RL algorithms and solved problems, based on the OpenAI Gymnasium module.

## Files

  q_learning: contains Q-learning method
  
  dynamic_programming: contains the dynamic programming methods, policy
            iteration and value iteration algorithms.
     
  monte_carlo: contains various Monte Carlo methods
  
  Taxi-v3: GridWorld environment solved by both policy iteration and value iteration.
  
  FrozenLake-v1: GridWorld environment solved by Q-learning.

  Blackjack-v1: Card game, solved by Monte Carlo methods
  
  Easy21: Environment of a simplified version of Blackjack

  Easy21Agent: Contains an Easy21Agent class which includes various Monte-Carlo policy evaluation and control RL algorithms
